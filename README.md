# MatchingUs


## 프로젝트 소개


<aside>
💡 MATCHING US는 대학생들을 위한 캠퍼스 매칭 프로그램 입니다.

</aside>


소개팅 & 과팅이 하고 싶은데 주선자가 필요하시다면, MATCHING US에서 주선자 역할을 해드릴게요! 

 MATCHING US를 통해 캠퍼스에서 여러분의 인연을 만나보세요!
